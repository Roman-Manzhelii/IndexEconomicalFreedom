# Index of Economic Freedom

This project is a transparent, reproducible attempt to build a country-level economic freedom index from public international data.

## Project idea

The goal is to assemble a defensible raw dataset first, then gradually turn it into an index.

The index is expected to cover several broad dimensions:
- rule of law,
- government size,
- regulatory efficiency,
- open markets and monetary stability.


## Files currently available

The following source files are already in the project root:

### Worldwide Governance Indicators

- `wgidataset_with_sourcedata-2025.xlsx`

This file is expected to provide governance indicators such as:
- `RL.EST` for rule of law,
- `CC.EST` for control of corruption,
- `RQ.EST` for regulatory quality.

### World Development Indicators

- `WDICSV.csv`
- `WDICountry.csv`
- `WDIcountry-series.csv`
- `WDISeries.csv`
- `WDIseries-time.csv`
- `WDIfootnote.csv`

These files are expected to provide the remaining macro and policy indicators, together with country metadata, series metadata, and notes.
